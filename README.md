# 📈 Stock Price Forecasting System
AI-Powered Stock Prediction using LSTM & XGBoost Models

## 🎯 What This Project Does

This system predicts stock prices and trends using:

- 🔹 LSTM (Deep Learning for time-series)
- 🔹 XGBoost (Fast & accurate ML model)
- 🔹 Model comparison & evaluation

👉 It helps users understand:
- Future stock price trends  
- Model performance  
- Data-driven investment insights  

---

## 📊 Project Highlights

| Feature | Description |
|--------|------------|
| Dual Models | LSTM + XGBoost for prediction |
| Forecasting | Future stock price prediction |
| Visualization | Interactive charts using Plotly |
| Evaluation | MAE, RMSE, MAPE metrics |
| Dashboard | Web-based UI using Flask |

---

## 🏗️ System Architecture

```
Stock Data → Data Processing → Feature Engineering → Models (LSTM & XGBoost)
         → Evaluation → Forecast Output → Web Dashboard
```

## 🚀 Quick Start

### 🔹 Prerequisites
- Python 3.8+
- pip

---

### 🔹 Setup

```bash
git clone https://github.com/yatharth123/Stock-Forecasting-System.git
cd Stock-Forecasting-System

pip install -r requirements.txt
python app.py
```

👉 Open browser:
```
http://127.0.0.1:5000/
```
